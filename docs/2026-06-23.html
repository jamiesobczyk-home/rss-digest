Hacker News · 1:05 AM

The new HTTP QUERY method explained

RFC 10008 introduced the HTTP QUERY method to address limitations when performing complex database queries—traditional GET requests with query parameters create unwieldy URLs and encoding issues, while POST requests introduce semantic problems since they're non-idempotent and intended for resource creation, complicating automatic retries and caching. QUERY resolves this by allowing request bodies while maintaining proper HTTP semantics for safe, read-only operations.

Who Does What? Team Topologies for the Agentic Platform

An agentic platform requires clarifying not just what capabilities are needed, but who provides them—the role of Team Topologies.
