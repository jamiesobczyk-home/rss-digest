Cisco Finesse Remote File Inclusion Vulnerability

Cisco Finesse contains a remote file inclusion vulnerability that allows unauthenticated attackers to load arbitrary files by tricking users into clicking malicious links, potentially enabling browser-based attacks and arbitrary code execution. The flaw results from insufficient validation of user-supplied input in HTTP requests. Cisco has released software updates to address the vulnerability (CVE-2026-20175); no workarounds are available.
