Armadillo – A DNS Server in Gleam for Homelab Use

Armadillo is a self-hosted DNS server written in Gleam for homelab networks that resolves local domains after being configured once on a router. It checks a local zone file for matching records first, then forwards unmatched queries to a configurable upstream resolver and caches responses by TTL. The server is available as a container image and includes a web UI and REST API for management.

Wayfinder Router: deterministic routing of queries between local and hosted LLM

Wayfinder Router is a CLI tool that routes prompts between local and cloud LLM models using deterministic structural analysis rather than model calls to decide. It scores prompt complexity by examining structure (length, headings, lists, code) and wording patterns in microseconds, completely offline, directing simple queries to cheaper local models and complex ones to expensive cloud models. Unlike other routers that rely on model calls or trained classifiers, Wayfinder eliminates routing latency and cost by making the same decision every time based on structural features.
