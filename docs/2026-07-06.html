When AI Costs More Than the Engineer

Anthropic spends $2 million per employee annually on compute—2.3x its payroll—while the software industry's top 1% spend only $89k per engineer on AI and the median spends $137. Three scenarios bracket how this gap closes by 2029: a bear case with token deflation, a base case where top-tier growth tapers, and a bull case where the rest of the market reaches Anthropic's spending ratio, driving AI costs to $596k per engineer by 2029. Hacker News · 8:35 PM

The Sneakerweb

Sneakerweb is a peer-to-peer protocol for web publishing that bypasses traditional DNS servers and domain registrars by storing websites directly on user devices and transferring them via physical media (.snk files). Users can view and manage offline website collections with a command-line tool that supports publishing, importing, exporting, and local browsing through a standard web browser. The system is built on the Willow protocol to prevent forgery, merge updates, and create compact files.
