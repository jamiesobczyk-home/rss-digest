A critical authentication bypass vulnerability in Cisco Catalyst SD-WAN Controller, Manager, and Validator allows unauthenticated remote attackers to bypass authentication and gain administrative privileges by exploiting a flawed peering authentication mechanism. Successful exploitation enables attackers to access NETCONF and manipulate the SD-WAN fabric's network configuration.
